HTML Interview Questions and Answers

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages using elements such as headings, paragraphs, images, and links.

2. What are tags in HTML?

Tags are used to mark up the start and end of an HTML element. Most tags come in pairs and define how the content inside them should be displayed.

3. What is the difference between div and span?

Div is a block-level element used to group larger sections of content, while span is an inline element used to group small pieces of text within a line.

4. What is the purpose of the DOCTYPE declaration?

The DOCTYPE declaration tells the browser which version of HTML is being used. It helps the browser render the page correctly in standard mode.

5. What are semantic elements?

Semantic elements clearly describe their meaning in the code. Examples include header, footer, article, and section. They help improve code readability and accessibility.

6. What is the difference between id and class?

Id is a unique identifier for one element, while class can be used for multiple elements. Both are used for styling and scripting.

7. What are empty elements?

Empty elements are HTML elements that do not have any content between the opening and closing tag. Examples include br, hr, and img.

8. How do you add a link to a web page?

You use the anchor tag with the href attribute. The href specifies the URL, and the text between the tags is clickable.

9. What are HTML entities?

HTML entities are special codes used to display reserved characters like less than, greater than, ampersand, and symbols like copyright.

10. What is the use of the iframe element?

The iframe element is used to embed another webpage or content such as a video or a map into the current page.
